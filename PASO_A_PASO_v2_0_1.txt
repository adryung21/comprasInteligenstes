ACTUALIZAR GITHUB
1. Descomprime el ZIP de corrección rápida.
2. Abre el repositorio comprasInteligenstes.
3. Add file → Upload files.
4. Sube los archivos y reemplaza los anteriores.
5. Pulsa Commit changes.
6. Abre:
   https://adryung21.github.io/comprasInteligenstes/?v=201
7. Inicia sesión como administrador.
8. Espera el mensaje de reparación automática.
